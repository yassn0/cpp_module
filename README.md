# C++ Modules (00-09)

Ensemble complet des modules C++ du cursus 42, couvrant les fondamentaux de la programmation orientée objet en C++98.

### Fondamentaux
- Classes et objets
- Encapsulation et membres
- Constructeurs/destructeurs
- Allocation dynamique (new/delete)
- Pointeurs et références

### POO Avancée
- Héritage simple et multiple
- Polymorphisme et méthodes virtuelles
- Classes abstraites
- Forme canonique de Coplien
- Deep copy vs Shallow copy

### Fonctionnalités C++
- Surcharge d'opérateurs
- Exceptions (try/catch/throw)
- Cast (static, dynamic, reinterpret, const)
- Templates de fonctions et classes
- STL (containers, iterators, algorithms)

## Compilation

Chaque exercice contient un Makefile avec les règles :
```bash
make        # Compile
make clean  # Nettoie les .o
make fclean # Nettoie tout
make re     # Recompile
```

### Obligatoire
- ✅ Forme canonique de Coplien (modules 02-09)
- ✅ Destructeurs virtuels pour classes polymorphes
- ✅ Gestion mémoire sans leaks
- ✅ Include guards dans tous les headers
- ✅ Nommage : `UpperCamelCase` pour les classes


## Progression

```
00-02 : Bases du C++ et POO
03-05 : Héritage, polymorphisme, exceptions
06-09 : Cast, templates, STL et algorithmes
```
